WHTT SENSOR

WIRELESS HUMIDITY AND TEMPERATURE SENSOR
The WHTT Indoor Sensor is a low-power, long range, wireless sensor that provides 0.5 °C accurate temperature and 2 % RH humidity monitoring at every location. Our smart sensors make it possible to monitor 24/7 for up to 10 years without the need for maintenance.
Features
  • Temperature measurement range: -40° C to 85° C
  • 0.5 °C accurate temperature monitoring
  • Relative humidity measurement range: 0 to 100%
  • 2 % RH accurate humidity monitoring
  • Battery voltage: 3.6 VDC*
  • Battery life: 3 to 5 years*
  • 1 connector for external power supply
  • External power supply range: 5VDC
  • Dimensions (H x W x D) 86 x 86 x 25.5 mm
  • Wall or double-sided mounting
  • Easy mounting (screws / adhesive)
  • User Customized reporting Modes
  • Robust sensor with Internal antenna and external antenna
